Exploring Machine Learning in Healthcare Diagnostics

About This Project

This project aims to develop a machine learning model that predicts the likelihood of diabetes in individuals based on key medical parameters such as glucose level, BMI, blood pressure, age, and insulin levels. By identifying early indicators of diabetes, this project seeks to contribute to preventive healthcare decision-making.

Project Features

  • Multiple ML Models: Logistic Regression, Random Forest, XGBoost, and TensorFlow Neural Network
  • Comprehensive Evaluation: Accuracy, Precision, Recall, F1-Score, and ROC-AUC metrics
  • Model Interpretability: SHAP and LIME integration for feature importance
  • Web Interface: Flask-based application with real-time predictions
  • End-to-End Pipeline: From data preprocessing to model deployment

Dataset Information

Dataset: PIMA Indians Diabetes Database (Kaggle)
Size: 768 records with 9 columns (8 features and 1 target variable)
Features: Pregnancies, Glucose, Blood Pressure, Skin Thickness, Insulin, BMI, Diabetes Pedigree Function, Age
Target: Outcome (1 = Diabetic, 0 = Non-Diabetic)
