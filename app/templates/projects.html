Diabetes Prediction Model

Status: Active | Type: Capstone Project

An end-to-end machine learning project that predicts diabetes risk based on medical parameters. This project demonstrates the complete data science workflow from data collection to model deployment.

Key Features:

  • Multiple ML algorithms: Logistic Regression, Random Forest, XGBoost, Neural Networks
  • Comprehensive data preprocessing and feature engineering
  • Hyperparameter tuning using GridSearchCV
  • Model evaluation with multiple metrics (Accuracy, Precision, Recall, F1-Score, ROC-AUC)
  • Model interpretability using SHAP and LIME
  • Flask web application for real-time predictions

Technologies Used:

Python, Pandas, NumPy, Scikit-learn, XGBoost, TensorFlow, Matplotlib, Seaborn, Flask, SHAP, LIME, Jupyter Notebook

Results:

  • Achieved ROC-AUC score > 0.85 across multiple models
  • Successfully deployed web application with real-time prediction capability
  • Comprehensive model comparison and evaluation

Project Methodology

1. Data Collection & Exploration

Used the PIMA Indians Diabetes Dataset from Kaggle. Performed exploratory data analysis to understand data distributions, correlations, and identify potential issues.

2. Data Preprocessing

Handled missing values by replacing unrealistic zeros with NaN and using median imputation. Applied feature scaling using StandardScaler for optimal model performance.

3. Model Training

Trained four different models with hyperparameter tuning:

  • Logistic Regression: Baseline classification model
  • Random Forest: Ensemble method for non-linear relationships
  • XGBoost: Gradient boosting for higher predictive power
  • TensorFlow Neural Network: Deep learning for pattern recognition

4. Model Evaluation

Compared models using multiple metrics and visualizations including confusion matrices, ROC curves, and classification reports. Selected best model based on ROC-AUC score.

5. Deployment

Integrated the best-performing model into a Flask web application with multiple pages including homepage, resume, projects, and diabetes prediction interface.
